# XDR 구축 제안서

## 1. 개요

### 1.1 프로젝트 배경
- 현재 SIEM/SOAR 기반 보안 운영 체계 운영 중
- 엔드포인트 및 네트워크 가시성 부족
- 크로스 레이어 상관분석 필요성 대두

### 1.2 목표
- EDR, NDR 통합을 통한 XDR 플랫폼 구축
- 통합 위협 탐지 및 대응 체계 확립
- 평균 탐지 시간(MTTD) 및 대응 시간(MTTR) 단축

## 2. 현황 분석

### 2.1 현재 보안 인프라
```
+------------------+     +------------------+
|      SIEM        |<--->|      SOAR        |
| (로그 수집/분석) |     | (자동화 대응)    |
+------------------+     +------------------+
         ^
         |
    [로그 수집]
         |
+--------+--------+
|   서버/애플리케이션   |
+------------------+
```

### 2.2 Gap 분석
| 영역 | 현재 상태 | 목표 상태 | Gap |
|------|-----------|-----------|-----|
| 엔드포인트 | 안티바이러스만 운영 | EDR 기반 행위 탐지 | EDR 도입 필요 |
| 네트워크 | 방화벽 로그만 수집 | NDR 기반 트래픽 분석 | NDR 도입 필요 |
| 상관분석 | 단일 소스 기반 | 크로스 레이어 분석 | 통합 룰 개발 필요 |

## 3. 제안 아키텍처

### 3.1 XDR 통합 아키텍처
```
                    +---------------------------+
                    |      XDR 플랫폼           |
                    |  (통합 대시보드/분석)     |
                    +---------------------------+
                               ^
           +-------------------+-------------------+
           |                   |                   |
    +------+------+     +------+------+     +------+------+
    |    SIEM     |     |    EDR      |     |    NDR      |
    | (로그분석)  |     | (엔드포인트)|     | (네트워크)  |
    +-------------+     +-------------+     +-------------+
           ^                   ^                   ^
           |                   |                   |
    +------+------+     +------+------+     +------+------+
    | 서버/앱로그 |     |  에이전트   |     | 네트워크탭  |
    +-------------+     +-------------+     +-------------+
```

### 3.2 데이터 흐름
1. 각 소스에서 데이터 수집
2. 정규화 및 enrichment
3. 크로스 레이어 상관분석
4. 알림 생성 및 자동 대응

## 4. 구현 계획

### 4.1 Phase 1: EDR 도입
- EDR 솔루션 선정 및 PoC
- 에이전트 배포 계획 수립
- SIEM 연동 테스트

### 4.2 Phase 2: NDR 도입
- NDR 솔루션 선정 및 PoC
- 네트워크 탭/미러링 구성
- SIEM 연동 테스트

### 4.3 Phase 3: 통합 및 최적화
- 크로스 레이어 상관분석 규칙 개발
- SOAR 플레이북 확장
- 대시보드 통합
